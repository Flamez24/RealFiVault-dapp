# RealFiVault DApp

Trustless micro-savings protocol on Pharos Mainnet.

**Contract:** `0x1c5b5b9dcb46d4ca16e8387419ba7f7c3f269ae2`  
**Network:** Pharos Pacific Ocean Mainnet (Chain ID: 1672)  
**Explorer:** https://pharosscan.xyz

---

## Run Locally

```bash
npm install
npm run dev
```

Open http://localhost:5173

---

## Deploy to Vercel

### Option 1 — Vercel CLI
```bash
npm install -g vercel
npm run build
vercel --prod
```

### Option 2 — Vercel Dashboard
1. Push this folder to a GitHub repo
2. Go to vercel.com → New Project
3. Import the repo
4. Framework: Vite
5. Deploy
